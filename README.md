# Minecraft, but I made it in 48 hours*.

##### Features:
- Infinite, procedurally generated world
- "Biomes"
- Full transparency + translucency support
- Sprite blocks (flowers)
- Animated blocks (water + lava)
- Distaince fog
- Blocks
  - Grass
  - Dirt
  - Stone
  - Sand
  - Water
  - Glass
  - Log
  - Leaves
  - Rose
  - Buttercup
  - Coal
  - Copper
  - Lava
  - Clay
  - Gravel
  - Planks
